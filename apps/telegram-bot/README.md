# my-telegram-bot

### создать новый репозиторий в командной строке

```sh
echo "# my-telegram-bot" > README.md
git init
git add README.md
git commit -m "first commit"
git branch -M main
git remote add origin https://git@git.sourcecraft.dev/hangovergordeev/my-telegram-bot.git
git push -u origin main
```

### или загрузить существующий репозиторий из командной строки

```sh
git remote add origin https://git@git.sourcecraft.dev/hangovergordeev/my-telegram-bot.git
git branch -M main
git push -u origin main
```
